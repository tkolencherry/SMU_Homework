Open Weather API

Source Info

The Open WeatherMap API is published by OpenWeatherMap, one of the leading digital weather information platforms. This project specifically used the Current Weather Data API to pull city specific information.

First, a random set of longitudes and latitudes was generated using the Numpy library. For each pairing of latitiudes and longitudes, the nearest city was found using CityPy. The iteration this analysis was based off of included 615 cities. The list of cities was then used to ping the Current Weather Data API from Open Weather to get different weather variables such as temperature, humidity, windspeed, cloudiness, and pressure. Since not all cities are included in the API, the sample of cities whittled down from 615 to 564. The cities were further filtered to account for only those with humidity less than 100%, resulting in 546 cities. These 546 cities are represented in each of the scatter plots included on this site.
